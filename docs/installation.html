Requirements

LAGOON-MCL uses Nextflow as a workflow manager and Singularity or Conda for the use of the different tools

Installation step (Singularity)

  1. Install Nextflow
  2. Install Singularity
  3. Download the pipeline

    4. Clone the current GitHub repository.


    git clone https://github.com/jroussea/LAGOON-MCL.git
    cd LAGOON-MCL
  4. Build containers

    5. You build container images from XXX.sif files.


    bash build_singularity_containers.sh
  5. Test the pipeline on a minimal dataset with a single command
    6. nextflow run main.nf nextflow.config -profile test,singularity

Installation step (Conda)

  1. Install Nextflow
  2. Install Conda (Anaconda or Miniconda)
  3. Download the pipeline

    4. Clone the current GitHub repository.


    git clone https://github.com/jroussea/LAGOON-MCL.git
    cd LAGOON-MCL
  4. Build containers

    5. You build conda environment from XXX.yaml files.


    bash build_conda_environment.sh
  5. Test the pipeline on a minimal dataset with a single command
    6. nextflow run main.nf nextflow.config -profile test,conda

Post-installation

It is possible that it is necessary to change the usage rights of the scripts present in the bin/ folder. To do this, you can run the following command:
cd bin/
chmod +x *
